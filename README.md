# Deskripsi Aplikasi
YourWish adalah aplikasi wishlist sederhana berbasis Flutter yang membantu pengguna mencatat, mengelola, dan memantau barang impian mereka.
Aplikasi ini dirancang dengan tampilan clean dan aesthetic sehingga mudah digunakan untuk kebutuhan sehari-hari. Pengguna dapat menambahkan wishlist baru, mengedit, menghapus, mencari dan memfilter wishlist berdasarkan kategori.
Pengguna dapat membuat akun melalui fitur registrasi dan masuk ke aplikasi menggunakan login yang terhubung dengan sistem autentikasi. Setelah berhasil masuk, pengguna dapat menambahkan wishlist baru, mengedit informasi wishlist, serta menghapus wishlist yang sudah tidak diperlukan. Aplikasi juga menyediakan fitur pencarian dan filter kategori agar pengguna dapat menemukan dan mengelompokkan wishlist dengan lebih mudah. Untuk meningkatkan kenyamanan penggunaan, aplikasi ini dilengkapi dengan dark mode dan light mode sehingga pengguna dapat menyesuaikan tampilan aplikasi sesuai preferensi mereka.
